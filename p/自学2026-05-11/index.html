📅 主线任务（注意要从实践出发，在实践中完善理论知识，做好资料整理）

近期目标：计划六月前完成linux学习和项目，七月边做简历边复习八股和刷代码随想录以及完善项目；英语继续精听，后续继续刷剑桥雅思，研二上必须考完；

  • 嵌入式: 买了一个项目，准备开始学习；学习一下vibe coding；

一、RK3588四路IPC项目（核心主线）

  1. 项目定位：工业级4路视频流+MPP硬件解码+多路YOLO检测融合+RTSP/GB28181推流监控项目，可改造为多模态智能IPC，适配小米Miclaw多模态岗位。
  2. 极简执行步骤：先完整跑通源码 → 定位图像帧数据 → 仅新增代码集成多模态/驱动，不改动原有功能。
  3. 关键概念
    • MPP：瑞芯微视频专用硬件，做H.264/H.265硬解码；NPU：AI推理硬件，跑YOLO、多模态大模型，二者分工独立。
    • OpenCV：C++图像工具库，用于画框、四路拼图、格式转换；Python版功能一致，但嵌入式项目受性能限制必须用C++版。
    • IPC：安防领域即网络摄像头，本项目等价于高性能智能IPC主机。
    • top命令：Linux系统监视器，排查CPU、内存、线程、内存泄漏、卡顿问题。

二、Vibe Coding & AI编程工具

  1. Vibe Coding：自然语言描述需求，AI实现代码细节，专注验收结果，高效改造嵌入式项目。
  2. 工具选型（最终敲定）
    • ✅ 首选：VSCode安装【Claude Code for VS Code（Anthropic官方正版）】，长上下文可读懂多文件嵌入式项目，支持DeepSeek API。
    • ❌ 放弃：终端CLI（新手无需使用）、Codex（适配简单脚本，底层硬件代码理解弱）、Claude Code YOLO（第三方魔改，自动乱改代码风险高）、其他杂牌套壳插件。
  3. 基础名词
    • CLI：命令行终端，黑框打字操作，多用于服务器/开发板批量操作。
    • JSON：通用配置文件格式，用于存储AI工具密钥、模型、接口地址。

三、下一步行动

  1. 编译运行四路IPC项目，验证监控、YOLO检测、推流功能正常。
  2. 安装官方Claude Code插件，定位项目图像帧，集成多模态大模型。
  3. 后续叠加MIPI摄像头、OLED驱动，打造求职差异化项目。
