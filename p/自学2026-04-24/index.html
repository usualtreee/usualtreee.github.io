一、核心考点词族联动（按语义分组，100%覆盖今日提问）

1. 字迹辨识度词族（legibility/legibly/legible）

词性 单词 核心释义 雅思高频用法 拼写&考点避雷
名词 legibility （字迹）清晰度、可辨识度 handwriting legibility 字迹辨识度 不可数名词，无复数形式
形容词 legible 字迹清晰可辨的 legible handwriting 清晰的字迹 前缀leg-，不要误写为lej-
副词 legibly （书写）清晰工整地 write legibly 书写工整 重音在第一音节/ˈledʒəbli/
反义名词 illegibility 字迹潦草、难辨认 illegibility of the script 答卷字迹潦草 否定前缀为il-，不是in-
反义形容词 illegible 字迹潦草难辨的 illegible script 潦草的答卷 同上
反义副词 illegibly 书写潦草不清地 write illegibly 写得潦草 同上
考点逻辑链：教孩子write legibly → 提升handwriting legibility → 阅卷老师可准确识别答卷 → 避免考试无故丢分

2. 特殊教育障碍词族（dyspraxia/dyslexia）

单词 音标核心差异 标准翻译 雅思场景核心特征 听力定位关键词
dyspraxia 结尾发音/ˈpræksiə/ 动作协调障碍 核心影响身体动作、精细运动控制，导致书写困难 motor skills/coordination/movement
dyslexia 结尾发音/ˈleksiə/ 诵读困难症（阅读障碍） 核心影响文字识别、拼写、读写能力，手写挫败感强 reading/spelling/letters/literacy
通用避雷：前缀均为dys-，不是dis-；均为不可数名词，无复数形式

3. 书写体核心词族（cursive/script/print）

单词 核心释义 雅思高频搭配 考点核心特征
cursive 连体手写体、连笔书写的 cursive handwriting 连体手写体 单词内字母连写，对精细动作要求高，学习难度大
print 印刷体、正体书写 print handwriting 印刷体 字母独立分开，易学习，适配低龄/有学习障碍的儿童
script 1. 考试答卷；2. 书写体体系；3. 戏剧剧本 exam script 考试答卷；cursive script 连体书写体 场景锁定：和exam搭配=答卷；和cursive搭配=书写体；和theatre搭配=剧本
官方同义替换：cursive = joined-up handwriting；script = answer paper/handwriting

4. 语言规范词族（punctuation/inconsistent）

单词 词性 核心释义 雅思高频搭配 考点核心
punctuation 不可数名词 标点符号；标点使用规范 spelling and punctuation 拼写与标点 雅思写作语法评分核心项，无复数形式
inconsistent 形容词 1. 用法不规范、前后不统一的；2. 与…矛盾的；3. 不稳定的 inconsistent with sth 与…相矛盾；inconsistent performance 表现不稳定 否定前缀in-，阅读判断题中与原文矛盾直接对应False选项
词族联动：inconsistent ↔ consistent（反义，一致的、连贯的）；inconsistency（对应名词形式）

二、高频短语&核心句式（全覆盖今日提问）

1. occurred to sb

  • 核心释义：（想法/事实）突然出现在某人脑海中，某人突然想到/意识到
  • 雅思高频句型：It never occurred to sb that + 从句（某人从未意识到…）
  • 真题原句：That aspect of handwriting had never occurred to me before.
  • 语法规则：主语必须是「事情/想法」，绝对不能是人；过去式必须双写r→occurred
  • 官方同义替换：dawn on sb / come to one’s mind / realise

2. fine motor skills

  • 核心释义：精细动作技能（手/手指小肌肉群的精准控制能力，儿童书写的核心基础）
  • 高频搭配：poor fine motor skills 精细动作能力差
  • 易混区分：gross motor skills 大动作技能（跑/跳/平衡），听力单选题高频干扰项
  • 官方同义替换：hand-eye coordination / manual dexterity

3. developing fluency isn’t any faster

  • 完整语境释义：（用键盘打字的方式）并不能丝毫更快地提升书写流利度
  • 核心语法：not any + 形容词比较级 = no + 形容词比较级，表强烈否定“丝毫没有更…”
  • 考点定位：25题核心干扰项，否定词isn’t弱读+失爆是听力高频踩坑点
  • 核心词补充：fluency此处特指书面表达/书写的流利度，而非仅口语流利度
  • 官方同义替换：is no faster / brings no improvement in speed

三、雅思听力易混词终极辨析（干扰项重灾区）

易混词对 核心区分 雅思高频场景
dyspraxia vs dyslexia 前者是身体动作控制障碍，后者是文字读写障碍 教育类Section3单选题核心干扰项
script vs handwriting script侧重完整文本/字体体系，handwriting侧重个人字迹风格/工整度 书写教学场景高频考点
cursive vs print cursive是字母连写的连体字，print是字母独立分开的印刷体 书写难度对比核心考点
inconsistent vs consistent 前者表否定“不一致、不规范”，后者表肯定“一致、连贯” 否定逻辑题核心，漏听in-直接选反答案
punctuation vs punctual 前者是标点符号，后者是形容词“准时的” 拼写高频错误，填空直接零分
