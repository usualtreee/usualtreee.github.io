Linux 驱动学习笔记 - 2026-04-23

1. 内核日志函数 printk

1.1 作用

  • 内核空间的 printf，输出到内核环缓冲区，通过 dmesg 查看。

1.2 日志级别

级别 宏 含义
0 KERN_EMERG 系统崩溃前的紧急消息
1 KERN_ALERT 必须立即处理
2 KERN_CRIT 严重错误
3 KERN_ERR 一般错误
4 KERN_WARNING 警告
5 KERN_NOTICE 普通但重要的通知
6 KERN_INFO 信息性消息（默认）
7 KERN_DEBUG 调试信息

1.3 使用示例 

1
2

printk(KERN_INFO "Driver loaded\n");
pr_info("Driver loaded\n");   // 推荐写法

1.4 注意事项

  • 不支持浮点数（%f 无效）
  • 指针需专用格式（如 %pK、%px）
  • 中断上下文中慎用，避免性能问题

{

  • includePath 数组内添加内核源码路径
  • 必须定义 __KERNEL__ 宏

3. 命令行技巧：cp -r 复制目录

3.1 基本用法 

1

cp -r 源目录 目标目录

3.2 常用参数

参数 作用
-r 递归复制（复制目录必备）
-i 覆盖前询问确认
-u 仅当源文件比目标新时才复制
-v 显示复制过程详细信息
-a 归档模式，保留所有属性

3.3 驱动开发场景 

1
2
3

cp 03_dev_t/ 04_cdev -r   # 备份上一版本
cd 04_cdev
make clean                # 清除编译产物，得到干净源码副本

4. 边缘 AI 推理：云端 vs. 端侧

4.1 对比

维度 云端推理 端侧推理
算力 近乎无限 受限
延迟 依赖网络 毫秒级响应
隐私 需上传数据 本地处理
可用性 需联网 可离线

4.2 RK3568 部署 Llama 方案

方案 说明
RKLLM 官方 NPU 加速（Toolkit + Runtime）
llama.cpp 社区 CPU 方案，无需 NPU

4.3 资源瓶颈

  • 推理是内存带宽密集型任务
  • RK3568 内存带宽约 8 GB/s，INT4 量化后 7B 模型约 3.5 GB
  • NPU 的核心作用是减少对系统内存带宽的依赖

5. RTOS 简介：Zephyr 与 NuttX

特性 Zephyr NuttX Linux
定位 功能丰富的物联网 RTOS 轻量级 Linux，POSIX 兼容 通用 OS
实时性 硬实时 硬实时 软实时
内存占用 50~300 KB 最低 32 KB Flash > 几 MB
许可证 Apache 2.0 Apache 2.0 GPLv2

❓ Q2：听力原文中 “hopefully” 是“希望”还是“谢天谢地”？

✅ 解答：

  • 意为 “希望”，表达愿望或期待。
  • 原文 “hopefully the queue won’t be too long” → 希望排队不会太久。
  • 不同于 Thank goodness（谢天谢地，庆幸已发生的事）。

补充例句：

  • 句子副词：Hopefully, we’ll arrive on time.（希望我们能准时到。）
  • 方式副词：The dog wagged its tail hopefully.（小狗满怀期待地摇着尾巴。）

❓ Q3：“swap ends” 是什么意思？

✅ 解答：

  • 意为 “交换场地 / 互换半场”，常见于足球、网球等体育语境。
  • 例句：Teams swap ends at half-time.（中场休息时两队交换场地。）

❓ Q4：post 和 stick 有什么区别？

✅ 解答：

维度 Post Stick
尺寸 大、粗、固定 小、细、可手持
典型物 门柱、路灯柱、邮筒 树枝、手杖、球棍
动词义 张贴、发布、邮寄 粘贴、插入、卡住
  • 听力实例：早期球门是 sticks（细木棍），现代是 goal post（坚固门柱）。
  • 动词区分：post a letter（寄信）vs stick a stamp（贴邮票）。

❓ Q5：go south 是什么意思？

✅ 解答：

  • 英语习语，意为 “变糟、下跌、失败、恶化”。
  • 例句：Things started to go south when the manager quit.（经理辞职后情况开始变糟。）

❓ Q6：that a shame 是什么意思？

✅ 解答：

  • 正确形式：That’s a shame / What a shame。
  • 意为 “真可惜、太遗憾了”，表达对小事不如意的惋惜。
  • 注意：shame 原意为“羞耻”，在此固定搭配中转为“遗憾”。
